# 🎮 AJIS-PS Management System

Sistem Informasi Rental PlayStation berbasis **Laravel** yang dikembangkan sebagai Projek Akhir mata kuliah **Pemrograman Web 2**. Sistem ini membantu pengelolaan operasional rental PlayStation mulai dari proses billing, checkout, manajemen member, transaksi kantin, hingga pencetakan struk.

## 🚀 Cara Menjalankan Project di local host

```bash
jalankan php di laragon
buka terminal jalankan 'cd rental-ps' diterminal laragon\www untuk mengakses folder laravelnya
jalankan 'php artisan serve' untuk menjalankan servernya
buka di web ketik http://127.0.0.1:8000/ maka akan masuk ke menu login
```
